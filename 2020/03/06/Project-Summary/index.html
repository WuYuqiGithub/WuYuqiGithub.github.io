Project Summary

2020-03-06

这是一款可以对板书笔记进行识别的APP。用户在首次使用该应用时需要进行注册，注册成功后则可登录至主页面。在主页面中，用户可以通过点击“上传图片”按钮或者“照片拍摄”按钮进行所需要识别图片的上传，上传完成后，用户可以在图片中选定所需要识别的区域，选择完成后点击“确认选取”按钮告知程序已经选区完成。此时程序会提醒用户已经选取的所需识别区域的数量，点击“开始识别”则可对选取区域进行识别，识别完成后会在图片下方的识别结果输出框中看见识别的结果。
